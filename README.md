<p>
  PTD is a full-featured React Material UI Admin Dashboard template. PTD is built with React,
  Redux & Material UI. We implemented all the features you might need to start a new Web
  application. The free version includes all Material UI components, Form elements, validation,
  JWT authentication, Sign in, sign up pages, Vertical navigation, Lazy loading, Code splitting.
</p>

<h2 id="availablescripts">Getting started</h2>

<p>In the project directory, you can run:</p>

<h3 id="npmstart"><code>npm run dev</code></h3>
<p>Runs the app in development mode. The page will reload if you make edits and show lint errors.</p>

<h3 id="npmrunbuild"><code>npm run build</code></h3>
<p>
Builds the app for production into the <code>build</code> folder.
It bundles React in production mode and optimizes the build for performance.
</p>
